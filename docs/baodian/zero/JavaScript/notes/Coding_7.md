## 1、输出下面代码结果

```js
let a = 3;
let b = new Number(3);
let c = 3;

console.log(a == b);
console.log(b === c);
console.log(a === b);
```

::: tip 参考答案
true false false

new Number() 是一个内置的构造函数。虽然它看起来像一个数字，但是它并不是一个真正的数字：它是一个对象，有额外的功能。所以使用 ==，只检查了它是否具有相同的值。而 === 时，还需要检测是否是同一类型。
:::
